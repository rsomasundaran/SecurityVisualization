Security Visualization

The following group of graphs plot variety of security relevant charts. They source security data from tcpdump/wireshark, ps, snort and firewall logs.

Learn more on interpreting charts »

Security Visualization Charts

No. of packets per protocol type

Chart Type: Bar
Source: tcpdump
X-axis: Source IP address
Y-axis: Number of packets per protocol type

This chart plots number of packets of various protocols such as TCP, UDP and ICMP from different source hosts. Plot

No. of blocked connections per day

Chart Type: Spline
Source: firewall log
X-axis: Day of the week
Y-axis: Count of blocked connections

This chart plots count of blocked connections at firewall on each day of the week. Data collected from firewall logs. Plot

Port activity at various times of the day

Chart Type: Scatter
Source: tcpdump
X-axis: Time of day
Y-axis: Destination port

This chart plots the count of packets comming to each destination port at different times of the day. Plot

Processes activity in the system

Chart Type: Column
Source: ps
X-axis: Process
Y-axis: %CPU usage

This chart plots the percentage of CPU time used per process (top 10) running on the system. Collected using 'ps'. Plot
